# 용어

## 개념

| 한글 | 영어 | 비고 |
|-|-|-|
| 잡 | job | cron에 의해 주기적으로 실행되는 일의 단위 |
| 태스크 | task | 잘게 나뉘어 queue에 넣어 실행되는 일의 단위 |

## 개발 환경 단계(stage)

| 용어 | 비고 |
|-|-|
| dev | 개발 중인 내용을 확인할 수 있는 환경 |
| alpha | 실제 환경과 유사하지만, 데이터가 분리된 환경 |
| beta | 실제 서비스 중인 데이터를 이용해서 개발된 로직을 테스트할 수 있는 환경 |
| production | 사용자들에게 서비스 되고 있는 환경 |
